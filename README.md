# CodeRSA

This is a rsa encryption/decryption tool in all platform with JVM.

## Usage

```
java -jar rsa*.jar $FLAG [$FILE_KEY $FILE_IN $FILE_OUT]

FLAG:
    k: generating key.
    e: encryption.
    d: decryption.
FILE_KEY: the public or private key file.
FILE_IN: input file
FILE_OUT: output file
```
